Form Input types Definition
Input type "text" This input type is used for single-line text input fields, where the user can enter any kind of text.
Input type "password" This input type is similar to the text input, but the text entered by the user is masked or hidden.
Input type "email" This imput type is used for email address input fields, and typically validates whether the entered text is properly formatted email address.
Input type "number" This input type is used for numberic input fields, where teh user can enter any kind of number, including decimals.
Input type "date" This input type is used for date input fields, allowing the user to select a data from a calendar or enter it manually
Input type "radio" This input type allows the user to select one option from a set of predefined option
Input type "checkbox" This input type allows the user to select one or more options from a set of predefined options
Input type "file" This input type allows the user to select one or more files to upload to the server.
Input type "submit" This input type is used for the submit button of a form, which triggers the form submission to the server.
